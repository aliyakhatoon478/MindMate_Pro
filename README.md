# MindMate Pro
**Full-Stack Mental Wellness Web Application**

MindMate Pro is a modern full-stack mental wellness application that enables users to track moods, reflect through journaling, and visualize emotional patterns via analytics dashboards.
The project demonstrates real-world application structure, reusable UI systems, protected routing, and data-driven frontend logic using a TypeScript-based stack.


## Project Overview

The goal of MindMate Pro is to create a calm, user-friendly wellness platform that encourages emotional awareness and consistency without being overwhelming.

The application focuses on:

* Daily mood tracking
* Emotional journaling
* Visual analytics
* Clean, scalable frontend architecture
* Full-stack integration readiness

This project is built to showcase **full-stack and frontend engineering skills** using modern tooling and best practices.

## Tech Stack

### Frontend

* **React** (with TypeScript)
* **Vite** (build tool)
* **React Router** for navigation
* **TanStack Query (React Query)** for state & data handling
* **Tailwind CSS**
* **shadcn/ui + Radix UI**
* Component-driven architecture
* Chart-based visualizations

### Backend

* **Node.js**
* **TypeScript**
* **Vite server runtime**
* Modular REST-style API structure

### Database & Schema

* **Drizzle ORM**
* Shared schema definitions
* Backend structured for easy database extension


## Authentication & Routing

* Login and Signup pages implemented
* Protected routes using a custom route guard
* Auth-ready architecture (token-based auth can be plugged in)
* Clean separation of public and private routes


## Core Features

### Mood Check-In

* Mood selection interface
* Intensity slider
* Calm, minimal UI
* Designed for one check-in per day logic


### Dashboard

* Central wellness overview
* Mood summaries
* Wellness suggestions
* Modular components for future intelligence


### Mood Analytics

* Visual charts for mood distribution
* Weekly emotional trends

### Emotional Journal

* Journal writing interface
* View past reflections

### Profile Page

* User-centric wellness overview
* Aggregated emotional insights
